ISSUE #12 - 2021-04-11

Farewell, my friends!

Is what I said to my colleagues on Friday! As this week was my last week at Karmarama (Accenture Interactive) there were some farewells. I'll be saying goodbye proper sometime in the summer when we're allowed out of the house again!

In my time there I introduced the team to using Gatsby and styled-components, used Prisma and the weird client that used render props (remember render props) for GraphQL queries. I also sat us all through Egghead videos to learn about React hooks.

We also created a super slick style guide using MDX (beta!) other things to note would be an interactive brocure using Preact and Lottie animations.

This week

Not a great deal from me this week, I have been playing around with Svelte and making some notes on it comparing some patterns to how you would do the same in React.

Svelte feels really simple, you write html, css and your JS in the same file and it's super fast with Vite!
